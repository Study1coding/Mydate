# Mydate
 设计一个自定义日期类（MyDate）：包含月(int类型)、日(int类型)和年(int类型) 三个字段；增加合适的构造函数既能创建默认日期(01/01/1900)也能创建用户指定日期；另增设接口SetDate()用于日期设置操作；对每个字段提供属性；提供接口Display()能按”月/日/年”格式显示日期（提示： 要防范创建或设置了无效日期，考虑大月、小月、闰年的判断）。
